Landing page moderna para un evento que combina la cultura automotriz con la gastronomía, diseñada con una estética premium, animaciones suaves y experiencia inmersiva.

## 🧠 Descripción

Este proyecto es una landing page para un evento de exhibición de autos y hamburguesas, creada con enfoque en diseño visual, animaciones y experiencia de usuario.

El objetivo es transmitir exclusividad, estilo y emoción mediante una interfaz elegante y moderna.

---

## ⚙️ Tecnologías utilizadas

- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion
- TypeScript

---

## 🎨 Características

- Diseño premium con estética automotriz
- Animaciones suaves con Framer Motion
- Secciones organizadas (Hero, Info, Invitación, Sponsors, CTA)
- Fondo dinámico y estilo visual elegante
- Totalmente responsive (mobile first)
- UI enfocada en experiencia inmersiva

---

## 📸 Secciones del proyecto

- 🚀 Hero impactante con imagen full screen
- 📍 Información del evento
- 💌 Invitación al público
- 🤝 Organizadores y patrocinadores
- 🔥 Call To Action final
